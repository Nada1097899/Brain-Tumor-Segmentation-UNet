#  Brain Tumor Segmentation using U-Net

An advanced Deep Learning project focused on identifying and segmenting brain tumors from MRI images using the U-Net architecture.

##  Project Overview
Image segmentation is crucial in medical diagnostics. This project implements a U-Net model to precisely delineate tumor regions in brain MRI scans, providing a high-accuracy automated tool for radiological analysis.

##  Tech Stack
* **Language:** Python
* **Deep Learning Framework:** TensorFlow / Keras
* **Architecture:** U-Net (Encoder-Decoder)
* **Libraries:** OpenCV, NumPy, Matplotlib, Scikit-learn.

## Methodology
- **Data Preprocessing:** Resizing and normalizing MRI images and their corresponding masks.
- **Model Architecture:** Implemented a symmetrical U-Net with skip connections to preserve spatial information.
- **Training:** Optimized using 'Adam' and evaluated with specialized loss functions for segmentation.

##  Results
The model successfully predicts tumor masks that closely align with the ground truth (Original Masks), demonstrating its ability to capture complex tumor shapes and locations.
